# Overview  
TodoQuest는 일상적인 할 일을 퀘스트로 제공하여 퀘스트를 클리어하는 Todo 모바일 애플리케이션입니다. 사용자는 퀘스트를 수행하며 경험치와 레벨을 올리고, 다양한 칭호와 배지를 획득할 수 있습니다. 목표 달성 동기 부여와 생활의 질 향상을 원하는 모든 사용자를 위한 서비스입니다.

# Core Features  
- 퀘스트 시스템: 일일/주간/월간/특별 퀘스트 제공, 맞춤형 퀘스트 추천, 난이도별 분류(초보자~전문가)
- 레벨 및 경험치 시스템: 퀘스트 완료 시 경험치(XP) 획득, 카테고리별 레벨, 성장 시각화
- 칭호/배지 보상 시스템: 퀘스트 달성 시 칭호 부여, 컬렉션 관리
- 카테고리 시스템: 건강, 운동, 자기계발 등 다양한 카테고리 지원 및 관심 카테고리 설정
- 프로필/마이페이지: 카테고리별 경험치/레벨, 계정 관리
- 통계/리포트: 주간/월간 달성률, 카테고리별 통계, 개인화 리포트

# User Experience  
- 주요 페르소나: 자기계발에 관심 있는 직장인, 학생, 습관 형성이 필요한 일반 사용자
- 주요 플로우:
  1. 소셜 로그인(Google/Apple) 및 관심 카테고리 선택
  2. 메인 화면에서 추천/진행 중 퀘스트 확인 및 관리
  3. 퀘스트 상세에서 완료/삭제
  4. 퀘스트 달성 시 경험치/칭호 획득
  5. 마이페이지에서 성장 현황, 칭호 컬렉션, 통계 확인
- UI/UX 고려사항: 직관적 네비게이션, 성장 시각화, 애니메이션, 접근성

# Technical Architecture  
- 시스템 구성: Flutter(크로스플랫폼 모바일), Supabase(백엔드/DB)=
- 데이터 모델:
  - users: 사용자 정보, 전체 레벨, 소셜 로그인
  - quests: 퀘스트 정보, 난이도
  - categories: 카테고리 정보
  - quest_categories : 퀘스트의 카테고리
  - user_quests: 사용자별 퀘스트 진행 현황
  - titles: 칭호 정보
  - user_category_levels: 카테고리별 사용자 레벨/경험치
  - user_titles: 사용자별 칭호 보유 현황
  - quest_completion_logs: 퀘스트 완료 기록
  - user_interests: 관심 카테고리
- 주요 API:
  - 퀘스트 관리(생성/수정/삭제/조회)
  - 사용자 프로필/레벨/경험치
  - 칭호/배지 시스템
  - 카테고리 관리
  - 통계/리포트
- 인프라: Supabase(PostgreSQL), 서버리스 함수, OAuth 인증(Google/Apple)

# Development Roadmap  
- Phase 1: MVP
  - 소셜 로그인(Google, Apple)
  - 기본 퀘스트 시스템 및 카테고리 선택
  - 기본 레벨/경험치 시스템
  - 메인 화면 및 퀘스트 관리 UI
- Phase 2: 기능 확장
  - 맞춤형 퀘스트 추천 알고리즘
  - 확장된 칭호/배지 시스템
  - 카테고리별 레벨 관리
  - UI/UX 개선(다크 모드, 애니메이션)
- Phase 3: 고급 기능
  - 소셜 기능(친구, 공동 퀘스트)
  - 고급 캐릭터 성장 시스템
  - 개인화된 분석 리포트
  - 외부 서비스 연동

# Logical Dependency Chain
- 1. 사용자 인증 및 데이터베이스 설계(Users, Categories)
- 2. 퀘스트/카테고리/칭호 데이터모델 및 API 구현
- 3. 메인 화면(퀘스트 관리) 및 프로필/레벨 화면 UI
- 4. 경험치/레벨/칭호 시스템 연동
- 5. 추천 알고리즘 및 통계/리포트 기능 추가
- 6. 소셜/고급 기능 및 외부 연동

# Risks and Mitigations  
- 사용자 참여도: 단순 반복적 퀘스트로 인한 흥미 감소 → 다양한 난이도/유형, 주기적 콘텐츠 업데이트
- 추천 알고리즘 정확도/성능: 단계적 개선, 효율적 DB 설계, A/B 테스트
- 리소스 제약: 핵심 기능 우선 구현, 모듈식 설계, 오픈소스/서드파티 활용
